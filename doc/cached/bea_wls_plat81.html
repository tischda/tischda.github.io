BEA WebLogic 8.1: Overview  

Course Number

PLAT-01-81

Overview

This course provides a 1-day technical introduction to the BEA WebLogic Platform 8.1 with a focus on BEA WebLogic Workshop 8.1.You will be able to make planning and design decisions that incorporate BEA WebLogic Workshop 8.1 and BEA WebLogic Platform 8.1.This course features BEA Ed.Lab, a revolutionary learning environment offered exclusively by BEA. BEA Ed.Lab eliminates the gap between technical training and real-world implementation.

Skills You Will Learn

  • Describe the methodology used by BEA WebLogic Workshop 8.1 to build, extend, integrate, deploy, and manage applications as end-to-end business processes
  • Identify where BEA WebLogic Platform 8.1 can be used to develop and integrate solutions within your IT environment
  • Make planning and design decisions that incorporate BEA WebLogic Workshop 8.1 and BEA WebLogic Platform 8.1

Who should attend this course

  • Developers
  • Administrators
  • Architects
  • Data Specialists
  • Managers
  • Executives

Format

Instructor-Led Training or Blended/eLearning

Topics Covered

  • BEA WebLogic Platform configuration and installation
  • JSPs, Page Flow, Struts, NetUI
  • Code completion, integrated debugging, tag libraries, instant deployment
  • Custom and embedded Java controls
  • Web services, asynchrony, loose coupling
  • XMLBeans, XQuery, Java-data manipulation
  • Portal development, layout, look and feel
  • User management, profiling, personalization, content management
  • Workflows, human work lists, messaging
