Fundamentals of J2EE Web Application Development Using BEA WebLogic Server 7.0

Course Number

WLS-D11-70

Overview

This 5 day course will focus on the flexibility afforded by using BEA WebLogic Server 7.0 and its implementation of the Java 2 Enterprise Edition (J2EE) specification during Web-based application development. Students will learn the core features of the presentation components (Servlets and JavaServer Pages) and service technologies (naming services, distributed transactions, and multi-tier database access). The course also covers the deployment of Web applications, using XML-based configuration files and the BEA WebLogic Server Administration Console. Additionally, students are introduced to the concepts of design patterns and how they are used in the Model-View-Controller (MVC) design architecture.

Students will get a brief introduction into the motivation and client-side use of Enterprise JavaBeans (EJB). Please note that EJB development and deployment will be deferred to the follow-on course: Developing Business Logic Components with Enterprise JavaBeans Using BEA WebLogic Server 7.0 (WLS-D21-70).

Instructor-led lab exercises will reinforce the syntax and architecture concepts presented in lecture through the incremental development of a simple online product purchase application.

Key Benefits

  • Understand the BEA WebLogic Server 7.0 implementation of the Java 2 Enterprise Edition Specification and how it allows for the development of modularized, easily maintainable code.
  • Develop a Web Application using J2EE component and service technologies, design patterns and the Model-View-Controller design architecture.
  • Gain knowledge of the syntax, configuration, and motivation behind the following J2EE technologies: Servlets v2.3, JSP v1.2, JNDI, JDBC, JTA, Web applications, Enterprise Applications, Client-side use of EJB.
  • Learn the basics of installation and monitoring of BEA WebLogic Server 7.0.

Audience

By providing an introduction to J2EE and WebLogic Server 7.0, the WLS-D11-70 class targets the J2EE-specified roles of Application Assembler and Deployment Specialist within the context of Web Applications.

  • An Application Assembler is responsible for developing Web-based presentation components and linking them to the business logic components.
  • A Deployment Specialist is responsible for configuring and deploying all component and service resources for Web Applications to the WebLogic Server 7.0 environment

Prerequisites

Completion of the course Object-Oriented Programming in Java (JAVA-D11) or equivalent knowledge of the following Java 2 Standard Edition concepts:

  • Object-Oriented Concepts (classes, objects, interfaces, inheritance)
  • Core Java Syntax (keywords, variables, flow and branching constructs)
  • Exception Handling
  • CLASSPATH and packages
  • Multi-threading
  • Input/Output Streams
  • Principles behind Remote Method Invocation and Object Serialization
  • Collection Classes (in java.util package)
  • Client-server database access through the Java Database Connectivity (JDBC) application programming interface, including the ability to write SQL statements and process ResultSet objects
  • Recommended - basic understanding of XML document syntax

Course Length

5 days

Course Format

Lecture with computer-based labs

Course Content

  • Introduction to J2EE and WebLogic Server Core Services (CORE)
  • Core WebLogic Server Administration (CWLA)
  • Core JNDI Concepts (JNDI)
  • Core JDBC Concepts (JDBC)
  • Introduction to Web Applications (INWA)
  • Core Servlet Development (CSRD)
  • Java Server Page Development (JSPD)
  • Maintaining State in a Web Application (STATE)
  • Advanced Servlet Development (ASRD)
  • Servlets As EJB Clients (EJBS)
  • Introduction to Patterns (IPAT)
  • Controller and Presentation Patterns (CPPA)
  • Introduction to Transactions (INTX)
  • Java Transaction API (JTXA)
  • Introduction to Enterprise Applications (INEA)
